1. installation of node js is required. path should be set for the same.
2. installation of mongo db is requied and path should be set for the same.
3. in the project folder, open terminal and run the following commands:
	npm initi -y
	npm i express
	npm i mongoose
	npm i body-parser
4. to run the file, type in terminal:
	node app.js
5. open the "http://localhost:3000" in your web browser.
